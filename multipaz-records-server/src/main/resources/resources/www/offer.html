Pre-authorized credential offers

You are playing the role of Admin of State. You are helping a user to obtain a credential from the official records of State. Select the record type and the record instance (if there are multiple records of the given type). Then generate a pre-authorized credential offer that the user then can use on their device to obtain the actual credential. Note that pre-authorized credential offers are very sensitive and can be converted to actual credentials by the bearer. Transaction code, typically passed to the user through a separate delivery method can add a little bit of additional protection.
